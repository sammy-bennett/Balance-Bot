# Balance Bot

## ESP32 balancing robot

This bot is designed using a gryro reader and PID controller to hold its balance on a flat surface.
It's my first roject as a EEE student desinged to be a quick introduction before my control systems module.

### Parts used

- drv8833
- ESP32 DEV
- various capsitors to get ride of noise on the power rails 
- 5v buck converters
- mpu5060

### sofware used

- PlatformIO runnign Arduino framework.
